## Short about

## Full about

pi-optimiser is a menu-driven hardening and tuning tool for Raspberry Pi OS. It turns the usual post-install checklist into a guided workflow that helps users configure storage, security, networking, package bundles, display settings, services, and firmware or boot options without memorising dozens of flags or editing system files by hand.

The project is built for both newcomers and experienced operators. The default TUI makes setup approachable on a fresh Pi, while the CLI remains fully usable for remote rollouts, repeatable builds, and automation. Dry runs, per-task state tracking, backup journals, snapshots, undo, and hardware-aware preflight checks make it practical for real systems rather than just demos.
